
# App2Class - תוכנית פיתוח

## ✅ שלב 1: רישום, אישורים ואווטאר (הושלם)
- דף נחיתה עם בחירת תפקיד
- רישום תלמיד/הורה/צוות עם סטודיו אווטאר
- מנגנון אישורים היררכי
- חשבון אדמין (admin@il / 123456)
- RLS policies מתוקנות

## ✅ שלב 2: דאשבורדים ומנגנון התראות (הושלם)
- Layout עם Sidebar מותאם תפקיד
- דאשבורד ראשי עם סטטיסטיקות לצוות
- עמוד אישורים מלא עם פילטרים ואווטארים
- התראות (Badge) על אישורים ממתינים
- תיקון באג אישור משתמשים (RLS policy)

## ✅ שלב 3: סביבת תלמיד - מבנה ראשוני (הושלם)
- דאשבורד תלמיד עם AI Insight, סטרייק, ספירה לאחור למבחנים
- תיקיות מקצועות (Subject Hubs) עם ממוצע דינמי ומגמות
- מרכז משימות עם מיון לפי סטטוס ודחיפות
- לוח שנה עם שכבות (מבחנים, אירועים, חופשות)

## ✅ שלב 4: סביבת מורה מקצועי - מבנה ראשוני (הושלם)
- טבלאות DB: lessons, attendance, lesson_notes, assignments, submissions, focus_reports
- RLS policies מלאות לכל הטבלאות
- דאשבורד מורה עם סטטיסטיקות
- הקראת שמות חכמה עם אווטארים, נוכחות, הערות התנהגותיות
- מערכת יצירת משימות עם סוגי מטלות, משקל, תאריכי הגשה
- ניווט ייעודי לפי תפקיד (תלמיד/מורה/הנהלה)

## ✅ שלב 5: סביבת רכז שכבה - מימוש מלא (הושלם)
- טבלאות DB: grade_events, event_approvals, tutoring_sessions, tutoring_students, staff_meetings, meeting_attendees, grade_announcements
- RLS policies מלאות לכל הטבלאות
- דאשבורד רכז שכבה עם תובנות AI, סטטיסטיקות ופעולות מהירות
- לוח מבחנים ואירועים עם בדיקת התנגשויות AI
- מוניטור דופק שכבתי - התקדמות חוצה-מקצועות וכיתות
- ניהול תגבורים - יצירה, שיבוץ מורים וחדרים
- ישיבות צוות עם פרוטוקולים דיגיטליים
- הודעות שכבתיות עם טיוטות ופרסום
- Edge Function: grade-coordinator-ai לתובנות וניתוח התקדמות
- ניווט ייעודי לרכז שכבה

## 🔜 שלב 6: מערכת ציונים וסטטיסטיקות ⭐ (הבא)

### 6.1 דף ציונים לתלמיד (StudentGradesPage)
- תצוגת ציונים לפי מקצוע (מבחנים, בחנים, עבודות, מטלות)
- ממוצע משוקלל לכל מקצוע (לפי weight_percent מטבלת assignments)
- גרף מגמה (קו) - התקדמות ציונים לאורך זמן (recharts)
- סיכום כללי: ממוצע כללי, מקצוע חזק/חלש, מספר הגשות
- טבלאות קיימות: assignments + submissions (כבר קיימות!)

### 6.2 דף סטטיסטיקות למורה (TeacherGradesPage)
- סטטיסטיקות כיתתיות: ממוצע, חציון, סטיית תקן
- חלוקה לאחוזונים (גרף עמודות)
- זיהוי תלמידים חלשים (<60) וחזקים (>90)
- השוואה בין מטלות

### 6.3 דשבורד הנהלה/רכז (שדרוג GradeProgressPage)
- ממוצעים שכבתיים לפי מקצוע
- השוואת כיתות בתוך שכבה
- מגמות לאורך זמן

## 🔜 שלב 7: סטודיו משימות (Task Studio)

### 7.1 יצירת משימה ע"י המורה
- ממשק הזנת שאלות + תשובות (Q&A Editor)
- סוגי שאלות: אמריקאי, פתוח, השלם, נכון/לא נכון
- ייבוא מקובץ / AI שמחלץ שאלות מחומר שהועלה
- טבלת DB חדשה: task_questions (assignment_id, question_type, question_text, options jsonb, correct_answer, explanation, order_num)

### 7.2 מצבי תרגול לתלמיד
- 🃏 פלאשקארדס (Flip Cards)
- 📝 בוחן אינטראקטיבי (Quiz Mode)
- ✍️ כתיבה חופשית (Open Answer + AI Check)

## 🔜 שלב 8: משחק נחשים וסולמות 🐍🪜 (משחק ראשון)

### 8.1 מנוע המשחק
- לוח 10x10 (100 משבצות) עם נחשים וסולמות
- קוביה אנימטיבית (framer-motion)
- כל X משבצות = שאלה מבנק task_questions
- תשובה נכונה = מתקדם, שגויה = נחש!
- 1-4 שחקנים (מולטיפלייר עתידי via realtime)

### 8.2 אינטגרציה
- המורה בוחר משימה קיימת → המשחק טוען את השאלות
- ציון מחושב: אחוז תשובות נכונות + מיקום סיום
- הציון מוזן אוטומטית לטבלת submissions

---

## 🔜 שלב 9: בוט מנטור מותאם אישית 🤖

### 9.1 שדרוג הבוט הקיים (ai-tutor edge function)
- הבוט מכיר את הפרופיל האישי (ציונים, חוזקות, חולשות)
- ממליץ על תרגול מותאם
- מסוגל ליצור שאלות תרגול בזמן אמת
- מצב "הכנה למבחן" מבוסס חומר מקצוע

### 9.2 יצירת משחקים ע"י הבוט
- המורה מתאר בטקסט חופשי מה הוא רוצה
- הבוט יוצר JSON של שאלות ותשובות
- אוטומטית נטען למשחק שנבחר

---

## 🔜 שלב 10: משחקים נוספים + HTML Page 🎮

### 10.1 טיפוס הר / הצלת נסיכה / בן האש ובת המים
- שלבים עם שאלות בכל שלב
- ויזואליה ייחודית לכל משחק

### 10.2 Blank HTML Page
- המורה מדביק קוד HTML/JS
- AI עושה אופטימיזציה
- קליטת ציונים מדף HTML חיצוני → submissions

### 10.3 AI יוצר משחק חופשי
- המורה מתאר → AI יוצר מבנה משחק
- ניתוח תיקייה כיתתית / שאלות לייב → משחק אוטומטי
